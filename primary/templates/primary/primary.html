Информационная система «Поиск поставщиков продукции»

в стадии разработки
Приложение имеет следующую функциональность:

Приложение содержит {{supplier_list }} поставщиков из {{country_list}} стран по {{category_list}} категориям

Пользователь по запросу на странице "Поиск поставщика"находит производителя/поставщика заданного продукта из заданной страны.

Пользователь может просмотреть список производителей продукта по всем странам.

Имеется большой перечень предприятий, с описанием производимой продукции. По этому описанию и производится полнотекстовый поиск.

Администратор сайта может добавлять данные из xlsx файла через поле "Загрузки" и редактировать базу данных в "Admin".
Особенности регистрации и авторизации пользователя:

При регистрации обязательно надо подтвердить email. Для подтверждения, на указанный полльзователем адрес отправляется уведомление для подтверждения email. Срок действия уведомления 24 часа.

При регистрации сохраняются в приложении данные об IP пользователя и об оборудовании (компьютере, планшете, телефоне) с которого произведена регистрация.

Авторизация пользователя с неподтвержденным email или заходящего с другого оборудования запрещена.
